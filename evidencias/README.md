# Pruebas de Software - Sistema Login

## Descripción
Este proyecto contiene evidencias y documentación de pruebas funcionales realizadas a un sistema de inicio de sesión (login).

## Objetivo
Verificar el correcto funcionamiento del sistema de autenticación mediante diferentes casos de prueba.

## Casos de prueba realizados
- Inicio de sesión exitoso
- Contraseña incorrecta
- Campos incompletos

## Evidencias
Las capturas de pantalla se encuentran en la carpeta `evidencias`.

## Herramientas utilizadas
- Python
- Visual Studio Code
- GitHub
